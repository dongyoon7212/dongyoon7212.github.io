# 코딩테스트 준비
백준 11654번


- - -
##### 문제
알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.

- - -
##### 입력
알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

- - -
##### 출력
입력으로 주어진 글자의 아스키 코드 값을 출력한다.

- - -
##### 예제 입력1
```
A
```

##### 예제 출력1
```
65
```
- - -
##### 예제 입력2
```
C
```

##### 예제 출력2
```
67
```
- - -
##### 예제 입력3
```
0
```

##### 예제 출력3
```
48
```
- - -
##### 풀이

이 문제의 경우 매우 간단하다.

ord를 이용하면 된다.
ord : 아스키코드를 반환

참고로 chr도 있다.
chr : 숫자에 맞는 아스키코드를 반환

- - -
##### 정답
```
a=input()
print(ord(a))
```

예제를 입력해보면
```
A
65
```

```
C
67
```

```
0
48
```
